

# Introduction

Montecarlo tree search의 예제를 보여줍니다. 
데모로서 OpenAI의 택시문제를 풉니다. 
어떻게 움직여야 하는지, 승객이 어디 있는지, 목적지가 어디인지 아무것도 가르쳐주지 않습니다. 
단순히 목적지에 데려다주면 점수를 받고, 잘못된 승객을 태우면 점수를 잃도록 만들었습니다.
